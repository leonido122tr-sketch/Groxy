# Исходный файл иконки

Поместите сюда ваш исходный файл изображения иконки и назовите его `icon-source.png`.

**Требования к файлу:**
- Формат: PNG (предпочтительно) или JPG
- Рекомендуемый размер: минимум 512x512px (лучше 1024x1024px или больше)
- Изображение должно быть квадратным (1:1)
- Фон должен быть темным (#0A0A0A) или прозрачным

**После размещения файла запустите:**
```bash
node scripts/generate_icons_from_source.js
```

Или укажите другой путь к файлу:
```bash
node scripts/generate_icons_from_source.js путь/к/вашему/файлу.png
```

Скрипт автоматически создаст все необходимые размеры иконок для Android.
